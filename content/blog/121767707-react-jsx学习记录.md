<p>简介</p>

<h3 id="why-jsx">为什么使用 JSX？</h3>

<p>React 认为渲染逻辑本质上与其他 UI 逻辑内在耦合，比如，在 UI 中需要绑定处理事件、在某些时刻状态发生变化时需要通知到 UI，以及需要在 UI 中展示准备好的数据。</p>

<p> React 并没有采用将<em>标记与逻辑进行分离到不同文件</em>这种人为地分离方式，而是通过将二者共同存放在称之为“组件”的松散耦合单元之中，来实现<a data-link-icon="https://csdnimg.cn/release/blog_editor_html/release1.9.5/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=LA92" data-link-title="关注点分离" href="https://en.wikipedia.org/wiki/Separation_of_concerns" title="关注点分离">关注点分离</a>。</p>

<p>React <a data-link-icon="https://csdnimg.cn/release/blog_editor_html/release1.9.5/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=LA92" data-link-title="不强制要求" href="https://react.docschina.org/docs/react-without-jsx.html" title="不强制要求">不强制要求</a>使用 JSX，但是大多数人发现，在 JavaScript 代码中将 JSX 和 UI 放在一起时，会在视觉上有辅助作用。它还可以使 React 显示更多有用的错误和警告消息。</p>

<p></p>

<h1>元素渲染</h1>

<p>元素是构成 React 应用的最小砖块。</p>

<p>与浏览器的 DOM 元素不同，React 元素是创建开销极小的普通对象。React DOM 会负责更新 DOM 来与 React 元素保持一致。</p>

<h2 id="rendering-an-element-into-the-dom">将一个元素渲染为 DOM</h2>

<p>假设你的 HTML 文件某处有一个 <code>&lt;div&gt;</code>：</p>

<pre>
<code class="language-javascript">&lt;div id="root"&gt;&lt;/div&gt;</code></pre>

<p>我们将其称为“根” DOM 节点，因为该节点内的所有内容都将由 React DOM 管理。</p>

<p>仅使用 React 构建的应用通常只有单一的根 DOM 节点。如果你在将 React 集成进一个已有应用，那么你可以在应用中包含任意多的独立根 DOM 节点。</p>

<p>想要将一个 React 元素渲染到根 DOM 节点中，只需把它们一起传入 <a data-link-icon="https://csdnimg.cn/release/blog_editor_html/release1.9.5/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=LA92" data-link-title="ReactDOM.render()" href="https://react.docschina.org/docs/react-dom.html#render" title="ReactDOM.render()">ReactDOM.render()</a>：</p>

<pre>
<code class="language-javascript">const e = &lt;h1&gt;Hello,World!&lt;/h1&gt;;
ReactDOM.render(e,document.getElementById('root'));</code></pre>

<h2 id="updating-the-rendered-element">更新已渲染的元素</h2>

<p>React 元素是<a data-link-icon="https://csdnimg.cn/release/blog_editor_html/release1.9.5/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=LA92" data-link-title="不可变对象" href="https://en.wikipedia.org/wiki/Immutable_object" title="不可变对象">不可变对象</a>。<span style="color:#fe2c24;"><strong>一个元素就像电影的单帧：它代表了某个特定时刻的 UI。</strong></span></p>

<p>所以更新UI的方式就是创建一个新的元素并重新渲染。</p>

<p>例如：</p>

<pre>
<code class="language-javascript">function tick() {
  const element = (
    &lt;div&gt;
      &lt;h1&gt;Hello, world!&lt;/h1&gt;
      &lt;h2&gt;It is {new Date().toLocaleTimeString()}.&lt;/h2&gt;
    &lt;/div&gt;
  );
  ReactDOM.render(element, document.getElementById('root'));
}

setInterval(tick, 1000);</code></pre>

<p>这个例子会在 <a data-link-icon="https://csdnimg.cn/release/blog_editor_html/release1.9.5/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=LA92" data-link-title="setInterval()" href="https://developer.mozilla.org/en-US/docs/Web/API/WindowTimers/setInterval" title="setInterval()">setInterval()</a> 回调函数，每秒都调用 <a data-link-icon="https://csdnimg.cn/release/blog_editor_html/release1.9.5/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=LA92" data-link-title="ReactDOM.render()" href="https://react.docschina.org/docs/react-dom.html#render" title="ReactDOM.render()">ReactDOM.render()</a>。</p>

<p><img alt="" height="71" src="https://i-blog.csdnimg.cn/blog_migrate/b25aafb18d655ae206215f0699a828df.png" width="428" /></p>

<p><strong> React 只更新它需要更新的部分</strong></p>

<p>React DOM 会将元素和它的子元素与它们之前的状态进行比较，并只会进行必要的更新来使 DOM 达到预期的状态。</p>

<h1>组件 &amp; Props</h1>

<p>组件允许你将<span style="color:#fe2c24;"> UI 拆分为独立可复用的代码片段，</span>并对每个片段进行独立构思。</p>

<h2 id="function-and-class-components">函数组件与 class 组件</h2>

<p>定义组件最简单的方式就是编写 JavaScript 函数：</p>

<pre>
<code class="language-javascript">function Welcome(props) {
  return &lt;h1&gt;Hello, {props.name}&lt;/h1&gt;;
}</code></pre>

<p>你同时还可以使用 <a data-link-icon="https://csdnimg.cn/release/blog_editor_html/release1.9.5/ckeditor/plugins/CsdnLink/icons/icon-default.png?t=LA92" data-link-title="ES6 的 class" href="https://developer.mozilla.org/en/docs/Web/JavaScript/Reference/Classes" title="ES6 的 class">ES6 的 class</a> 来定义组件：</p>

<pre>
<code class="language-javascript">class Welcome extends React.Component {
  render() {
    return &lt;h1&gt;Hello, {this.props.name}&lt;/h1&gt;;
  }
}</code></pre>

<p>上述两个组件在 React 里是等效的。</p>

<h2 id="rendering-a-component">渲染组件</h2>

<p>之前我们所定义的都是React元素都是DOM标签；</p>

<p>React可以用户自定义组件；</p>

<pre>
<code class="language-javascript">const e = &lt;Hello name="LiHua"/&gt;</code></pre>

<p>结合之前的函数组件渲染。</p>

<pre>
<code class="language-javascript">function Welcome(props) {
  return &lt;h1&gt;Hello, {props.name}&lt;/h1&gt;;
}

const element = &lt;Hello name="LiHua" /&gt;;
ReactDOM.render(
  element,

<h2 id="extracting-components">提取组件</h2>

<p>将组件拆分为更小的组件。</p>

<p>将下列按组件拆分。</p>

<pre>
<code class="language-javascript">function formatDate(date) {
  return date.toLocaleDateString();
}

function Comment(props) {
  return (
    &lt;div className="Comment"&gt;
      &lt;div className="UserInfo"&gt;
        &lt;img
          className="Avatar"
          src={props.author.avatarUrl}
          alt={props.author.name}
        /&gt;
        &lt;div className="UserInfo-name"&gt;
          {props.author.name}
        &lt;/div&gt;
      &lt;/div&gt;
      &lt;div className="Comment-text"&gt;{props.text}&lt;/div&gt;
      &lt;div className="Comment-date"&gt;
        {formatDate(props.date)}
      &lt;/div&gt;
    &lt;/div&gt;
  );
}

const comment = {
  date: new Date(),
  text: 'I hope you enjoy learning React!',
  author: {
    name: 'Hello Kitty',
    avatarUrl: 'https://placekitten.com/g/64/64',
  },
};
ReactDOM.render(
  &lt;Comment
    date={comment.date}
    text={comment.text}
    author={comment.author}
  /&gt;,
  document.getElementById('root')
);
</code></pre>

<p>元素传入ReactDOM.render()中，调用comment组件，在传入props参数，在渲染进根节点中。</p>

<p>这里需要赋值，直接在外部定义赋值更清晰。</p>

<p>该组件用于描述一个社交媒体网站上的评论功能，它接收 <code>author</code>（对象），<code>text</code> （字符串）以及 <code>date</code>（日期）作为 props。</p>

<p>该组件由于嵌套的关系，变得难以维护，且很难复用它的各个部分。因此，让我们从中提取一些组件出来。</p>

<p>例如提取Avatar组件。</p>

<pre>
<code class="language-javascript">function Avatar(props) {
  return (
    &lt;img className="Avatar"
      src={props.user.avatarUrl}
      alt={props.user.name}
    /&gt;
  );
}</code></pre>

<p>事件处理</p>

<p>事件处理函数和传统的不同。</p>

<p>传统：</p>

<pre>
<code class="language-html">&lt;button onClick="test()"&gt;123&lt;/button&gt;</code></pre>

<p> React</p>

<pre>
<code class="language-html">&lt;button onClick={test}&gt;123&lt;/button&gt;</code></pre>

<p>除此之外，传统的HTML可以通关返回false的方式去阻止默认行为。</p>

<p>但在React中不能直接返回false阻止，它必须显示的使用preventDefault（）。</p>

<pre>
<code class="language-javascript">function ActionLink() {
  function handleClick(e) {
    e.preventDefault();
    console.log('The link was clicked.');
  }

  return (
    &lt;a href="#" onClick={handleClick}&gt;
      Click me
    &lt;/a&gt;
  );
}</code></pre>
