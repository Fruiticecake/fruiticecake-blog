<h3>Markdown 基础语法</h3>

<p><strong>段落与换行</strong><br />
段落之间用空行分隔，行尾添加两个空格实现换行。例如：</p>

<pre>
<code>第一行（末尾两个空格）  
第二行
</code></pre>

<p></p>

<p><strong>标题</strong><br />
使用 <code>#</code> 表示标题层级，从 <code>#</code>（一级）到 <code>######</code>（六级）：</p>

<pre>
<code># 一级标题
### 三级标题
</code></pre>

<p></p>

<p><strong>文字样式</strong></p>

<ul>
	<li>斜体：<code>*斜体*</code> 或 <code>_斜体_</code></li>
	<li>粗体：<code>**粗体**</code> 或 <code>__粗体__</code></li>
	<li>删除线：<code>~~删除线~~</code></li>
</ul>

<p><strong>列表</strong><br />
无序列表用 <code>-</code>、<code>*</code> 或 <code>+</code>：</p>

<pre>
<code>- 项目1
- 项目2
</code></pre>

<p></p>

<p>有序列表直接写数字：</p>

<pre>
<code>1. 第一项
2. 第二项
</code></pre>

<p></p>

<p><strong>链接与图片</strong><br />
链接：<code>[文本](URL)</code><br />
图片：<code>![描述](图片URL)</code></p>

<p><strong>代码</strong><br />
行内代码：<code>`代码`</code><br />
代码块：用三个反引号包裹（可指定语言）：</p>

<pre>
<code class="language-python">print("Hello")
</code></pre>

<p></p>

<h3>高级技巧</h3>

<p><strong>表格</strong><br />
用 <code>|</code> 和 <code>-</code> 对齐：</p>

<pre>
<code>| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| 数据1  | 数据2 | 数据3  |
</code></pre>

<p></p>

<p><strong>脚注</strong><br />
通过 <code>[^标签]</code> 添加说明：</p>

<pre>
<code>这是一个脚注[^demo].  
[^demo]: 脚注内容。
</code></pre>

<p></p>

<p><strong>任务列表</strong><br />
在 GitHub 等平台支持：</p>

<pre>
<code>- [x] 完成
- [ ] 未完成
</code></pre>

<p></p>

<p><strong>数学公式</strong><br />
行内公式：<code>$公式$</code><br />
块公式：<code>$$公式$$</code><br />
例如： $$ E=mc^2 $$</p>

<p><strong>转义字符</strong><br />
用 <code>\</code> 避免符号被解析，如 <code>\*</code> 显示为普通星号。</p>

<h3>工具与扩展</h3>

<ul>
	<li><strong>编辑器推荐</strong>：VS Code（安装 Markdown 插件）、Typora（所见即所得）。</li>
	<li><strong>扩展语法</strong>：部分平台支持流程图（如 Mermaid）、目录生成（<code>[TOC]</code>）。</li>
	<li><strong>导出功能</strong>：通过 Pandoc 转换为 PDF、HTML 等格式。</li>
</ul>

<p><strong>注意事项</strong><br />
不同平台（如 GitHub、Notion）可能对语法支持有差异，需测试兼容性。</p>
