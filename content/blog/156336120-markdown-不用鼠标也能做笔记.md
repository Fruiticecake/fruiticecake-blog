#### Markdown 基础语法

段落与换行 段落之间用空行分隔，行尾添加两个空格实现换行。例如：

```
第一行（末尾两个空格）  
第二行

```

标题 使用`#`表示标题层级，从`#`（一级）到`######`（六级）：

```
# 一级标题
### 三级标题

```

文字样式

- 斜体：`*斜体*`或`_斜体_`
- 粗体：`**粗体**`或`__粗体__`
- 删除线：`~~删除线~~`

列表 无序列表用`-`、`*`或`+`：

```
- 项目1
- 项目2

```

有序列表直接写数字：

```
1. 第一项
2. 第二项

```

链接与图片 链接：`[文本](URL)`图片：`![描述](图片URL)`

代码 行内代码：``代码``代码块：用三个反引号包裹（可指定语言）：

```
print("Hello")

```

#### 高级技巧

表格 用`|`和`-`对齐：

```
| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| 数据1  | 数据2 | 数据3  |

```

脚注 通过`[^标签]`添加说明：

```
这是一个脚注[^demo].  
[^demo]: 脚注内容。

```

任务列表 在 GitHub 等平台支持：

```
- [x] 完成
- [ ] 未完成

```

数学公式 行内公式：`$公式$`块公式：`$$公式$$`例如： $$ E=mc^2 $$

转义字符 用`\`避免符号被解析，如`\*`显示为普通星号。

#### 工具与扩展

- 编辑器推荐：VS Code（安装 Markdown 插件）、Typora（所见即所得）。
- 扩展语法：部分平台支持流程图（如 Mermaid）、目录生成（`[TOC]`）。
- 导出功能：通过 Pandoc 转换为 PDF、HTML 等格式。

注意事项 不同平台（如 GitHub、Notion）可能对语法支持有差异，需测试兼容性。
